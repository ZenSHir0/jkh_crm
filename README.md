# JKH CRM — Система управления заявками ЖКХ

Веб-приложение для управления заявками на обслуживание с ролевой моделью доступа. Разработано как дипломный проект.

## Роли и возможности

| Роль | Возможности |
|------|-------------|
| **Резидент** | Создание заявок, редактирование профиля, комментарии к заявкам |
| **Мастер** | Просмотр назначенных заявок, отслеживание баланса, комментарии |
| **Администратор** | Управление пользователями, назначение мастеров на заявки, бонусы и штрафы, смена ролей, блокировка аккаунтов |

## Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone git@github.com:ZenSHir0/jkh_crm.git
cd jkh_crm
```

### 2. Создать папку для загрузок

```bash
mkdir uploads
```

### 3. Создать файл переменных окружения

```bash
touch .env
```

Пример содержимого `.env`:

```env
GMAIL_USERNAME=email@gmail.com
GMAIL_PASSWORD=your_app_password
UPLOADS_DIR=./uploads
```

> **Примечание:** Почта и пароль нужны только для работы модуля подтверждения email и сброса пароля. Если не нужны можно написать skip skip как юзернейм и как пароль, остальной функционал работает без них.  
> Используйте **App Password** из настроек Google-аккаунта, не основной пароль.

### 4. Запустить через Docker

```bash
docker compose up
```

### 5. Открыть в браузере

```
http://localhost:8080/
```
